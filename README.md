## 📖 Overview

**SkyCast** transforms raw weather data into an interactive visual experience.  
Built with **vanilla HTML, CSS, and JavaScript**, it enhances real-time weather data with smooth animations and dynamic UI elements.

---

## ✨ Features

- 🌍 **Global Search** — Get accurate weather by city name  
- 📍 **Live Location** — Fetch weather using browser geolocation  
- 🌡️ **Real-Time Data** — Temperature, humidity, and wind speed  
- ☀️ **Sun Tracking** — Position updates based on sunrise & sunset  
- ⚡ **Weather Effects** — Rain and lightning animations  
- 🌙 **Dark Mode** — Smooth UI theme switching  
- 🎨 **Dynamic Backgrounds** — Changes based on weather conditions

## 🧑‍💻 Run Locally

### 📦 Prerequisites

- Modern web browser  
- Internet connection  

### ▶️ Installation

```bash
git clone https://github.com/your-username/SkyCast.git
cd SkyCast
```
Open:

```bash
index.html
```

#### ⚙️ Technical Specs
- Weather API: Open-Meteo
- Geocoding: Open-Meteo Geocoding API
- Frontend: HTML, CSS, JavaScript
- Design: Responsive + animated UI

#### ⚠️ Notes
- Location access requires browser permission
- Internet connection required for API calls
